# Tesbih — Android &amp; iOS

Capacitor wrapper that turns the [tesbih](https://github.com/abourdim/tesbih) trilingual web app into a native **Android APK / AAB** and **iOS IPA**.

Based on the book **"تسبيح" (Tesbih)** by Sheikh Mohammed al-Ghazali (1917–1996).

## Quick start

```bash
./manage.sh    # auto-detects your OS, then opens the menu
```

Build, test on emulator, and prepare a Play Store / App Store release from a single guided menu. See [WORKSHOP.html](WORKSHOP.html) for the full step-by-step guide.

## Repo layout

| Item | Notes |
|---|---|
| `index.html` | Project landing page (download APK, links) |
| `WORKSHOP.html` | Full build / publish walkthrough |
| `capacitor.config.json` | App ID `org.workshopdiy.tesbih`, splash + statusbar configured |
| `package.json` | Capacitor 8.x + `@capacitor-community/text-to-speech` |
| `manage.sh` | OS dispatcher → `manage-{linux,macos,windows,ios}.sh` |
| `www/` | Built web app (synced from [tesbih](https://github.com/abourdim/tesbih) at build time) |
| `releases/` | Prebuilt artifacts (APKs / IPAs) |
| `store-assets/` | Play Store icon, feature graphic, privacy policy |
| `PLAY_STORE_LISTING.md` | Trilingual store copy (AR / EN / FR), ready to paste |
| `.gitignore` | Excludes `node_modules/`, `android/.gradle/`, keystores |
